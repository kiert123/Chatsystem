# Chatsystem
Client1.java is a user chat terminal designed with a cool dark cyberpunk matrix theme that features integrated emoji selection buttons for network messaging.

Client2.java is a side-by-side clone of the chat terminal that uses the exact same dark neon emerald design structure to communicate over the server network channels.

PreviousMessage.java is a persistent archive viewer window that cleanly displays past chat history logs read directly from your local tracking file.

ProjectSummary.java is a dedicated documentation window that outlines the structural file changes, theme adjustments, and dependencies of your system.
